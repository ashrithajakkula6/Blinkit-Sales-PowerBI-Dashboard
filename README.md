# Blinkit Sales Dashboard (Power BI)

## Project Overview
This project analyzes Blinkit grocery sales data using Power BI.

## Objectives
- Analyze sales performance
- Identify top selling items
- Evaluate item visibility impact on sales
- Analyze outlet performance

## Tools Used
- Power BI
- Excel / CSV
- Data Visualization

## Dashboard Insights
- Total Sales performance
- Sales by outlet
- Top selling products
- Relationship between item visibility and sales

## Files
Blinkit_Sales_Dashboard.pbix – Power BI dashboard  
blinkit_sales.csv – Dataset
